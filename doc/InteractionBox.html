Class: InteractionBox

InteractionBox(x, y, width, height, icon, hitFunction)

new InteractionBox(x, y, width, height, icon, hitFunction)

Permet de créer des objets interactibles qui active une action.
Parameters:
Name Type Description
x float La position en abcisse de la boîte de collision.
y float La position en ordonnée de la boîte de collision.
width float La largeur de la boîte de collision.
height float La hauteur de la boîte de collision.
icon sprite L'image de la boîte de collision.
hitFunction functon La fonction qui sera appelé quand la boîte de collision sera activée.
